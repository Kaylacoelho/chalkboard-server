# 🟫 ChalkBoard

A live sports scores + betting insights dashboard.

## Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Start the server
```bash
node server.js
```
Server runs at http://localhost:3001

### 3. Start your React app
In a separate terminal, from your React project folder:
```bash
npm run dev
```
Then copy `App.jsx` into your `src/` folder.

## Features
- ⭐ Best Bet of the Day
- ⚡ Upset Alerts
- ★ Favorite teams (pinned + persisted)
- 📈 Score timeline on live games
- 🔥 Momentum indicator
- Auto-refreshes every 30 seconds

## Leagues
NBA, NFL, NHL, MLS, UEFA Champions League
